# Parse sample corpus — bundled outputs

Local bundle of the **100 PDFs** from `manifest.json` (50 annual + 50 CFO), parsed with **open-parse** and **LiteParse**.

Generated by `0-work/scripts/run_parse_sample_bundle.sh`.

## Layout

```
parse-sample-corpus/
  manifest.json           # source key list (from pdf_page_counts.json)
  index.json              # per-document index with relative paths
  openparse-summary.json
  liteparse-summary.json
  pdfs/
    annual/{slug}.pdf
    cfo/{slug}.pdf
  openparse/
    annual/{slug}.json    # open-parse nodes + stats
    annual/{slug}.md      # text preview
    cfo/...
  liteparse/
    annual/{slug}.json    # LiteParse stats + page summaries
    annual/{slug}.md      # full markdown
    cfo/...
```

## Regenerate

```bash
cd 0-work/scripts
set -a && source .env && set +a
./run_parse_sample_bundle.sh
```
