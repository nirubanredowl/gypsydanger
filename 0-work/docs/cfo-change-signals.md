## Key finding

**There is no `CFO Appointment/Resignation` Markit tag.** Unlike CEO (`CEO/Managing Director - Appointment Resignation`, ~835 rows), CFO changes appear almost entirely as:

1. **Headline text** — "Appointment of CFO", "Resignation of Chief Financial Officer", etc.
2. **Tag:** `Company Administration - Other` (67% of CFO headlines)
3. **Secondary tags:** `Director Appointment/Resignation` (107), `Company Secretary Appointment/Resignation` (451) when CFO dual-hats

## Do not use for change detection

- `Change of Director's Interest Notice` (~102k) — trading/holdings notices
- Directors' reports without change headlines — context only, wrong event date
